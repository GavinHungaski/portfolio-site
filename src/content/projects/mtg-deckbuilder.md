Artifice Alley is a fullstack MTG companion app built from the ground up. It features card search powered by the Scryfall API, a full deck builder with persistent storage, and a playtesting simulator. The backend is an Express/Node REST API backed by PostgreSQL, and the frontend is a React SPA — all deployed and hosted live on Railway.
