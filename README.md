# Gavin Hungaski — Portfolio Site

Personal portfolio for Gavin Hungaski, a CS grad documenting his path from web developer to Robotics & ML software engineer — one project at a time.

Built with [Astro](https://astro.build), [Tailwind CSS](https://tailwindcss.com), and MDX.

## Features

- **Home** — hero, project showcase, and latest blog posts
- **Blog** — dev log with MDX support and RSS feed
- **About** — background and goals
- **Projects** — roadmap tracking progress toward robotics & ML

## Getting Started

```bash
npm install
npm run dev
```

| Command | Description |
|---|---|
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
